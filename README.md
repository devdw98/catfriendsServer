# 냐한친구들 (가제) 서버

### Spec
- Gradle Project
- Lang : Java 11
- SpringBoot : 2.3.2
- Packaging : Jar
- DB : MySQL (변동 가능)
- Dependencies (변동 가능)
  * Spring Web
  * Spring Session
  * Spring Data JPA
  * Lombok
  * OAuth2 Client

### 구현할 기능
 - 사용자관리
 - 결제 (구글 인앱 결제)
 - 우편함 (FCM)
 - 공지사항 (FCM)
 - 광고 시청 시 돈 2배
